COSA FARE PER FAR FUNZIONARE IL PROCESSO PER SINGOLO MODULO.

1- COPINCOLLARE IL PROPRIO CV NEL FILE DATA > CV_ESEMPIO
2- SELEZIONARE UN ANNUNCIO DA LINKEDIN E COPINCOLLARE LA PARTE DI REQUISITI E ATTIVITà PREVISTE NEL FILE DATA > ANNUNCIO_ESEMPIO
3- PRODURRE CON GEMINI 2.5 PRO 3-4 ESEMPIO DI DOCUMENTI PROGETTUALI DANDOGLI IN INPUT L'ANNUNCIO DI LAVORO. CHIEDERE DI FARE 3-4 ESEMPI DI DOCUMENTI PERTINENTI ALL'ANNUNCIO E INSERIRLI IN DATA > KNOWLEDGE_BASE
4- LA SENIORITY LEVEL è FORZATA IN UN PAIO DI FILE IN MODO MANUALE COME JUNIOR - MID. LASCEREI COSI
5- LANCIARE python -m analyzer.run_analyzer AVENDO CURA DI MODIFICARE NEL FILE analyzer > run_analyzer LA RIGA 14 HR_NEEDS INSERENDO 1/2 REQUISITI CHE IL SISTEMA CERCA IN MODO PRIORITARIO.
6- LANCIARE python -m analyzer.run_case_generator
7- LANCIARE python -m corrector.run_corrector_setup
8- LANCIARE python -m interviewer.main E INTERAGIRE CON IL CHATBOT DAL TERMINALE. PUOI FARGLI FINO A 3 DOMANDE PERTINENTI PER CHIARIRE. RISULTA LUNGO, CONSIGLIO COMUNQUE DI FARNE 1 PER INTERO E CON ATTENZIONE. ALTRIMENTI INIZIA A RISPONDERE USANDO CHATGPT.
9- LANCIARE python -m corrector.run_final_evaluation PER ATTIVARE LA CORREZIONE E CREAZIONE DEL REPORT FINALE DI ANALISI
10- LANCIARE python -m feedback_generator.run_feedback_generator PER ATTIVARE LA CREAZIONE DEL REPORT FINALE CON CORSI DI FORMAZIONE

COMANDI PER LANCIARE I MODULI

    python -m interviewer.main --> Lancia il chatbot
    python -m analyzer.run_analyzer --> attiva l'analisi del CV
    python -m analyzer.run_case_generator  --> attiva tutto il flusso di produzione dei case e di accomplishment criteria.
    python -m corrector.run_corrector_setup --> Attiva la generazione dei criteri di valutazione.
    python -m corrector.run_final_evaluation --> Avvia il processo di valutazione del case eseguito dai candidati.
    python -m feedback_generator.run_feedback_generator --> Genera il flusso di creazione report feedback + percorso formativo

LANCIA L'INTERA APP CON: streamlit run webapp/app.py
